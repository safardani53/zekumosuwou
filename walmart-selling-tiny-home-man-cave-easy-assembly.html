WALMART is selling a tiny home that's perfect for a "man cave."

Not many people would think to head to Walmart when looking for housing.

However, the big box store has started stocking several tiny homes, ranging from prebuilt sheds to cabin kits you can build yourself.

One of the many options on Walmart's website is the Cedarshed Studio Shed Kit, which is available in two sizes - a 9-foot-by-6-foot size for $4,928, and a bigger 12-foot-by-6-foot option for only $6,250.

The wood shed has two windows in the front and a door that opens halfway through which can used as another window.

The product description says the shed is easy to assemble and that it would make a great backyard "mancave" if you're not looking for another home, but rather just some extra space.
